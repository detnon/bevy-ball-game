# Bevy / Rust Tutorial 

Following the tutorials laid out by [Frederickjjoubert](https://github.com/frederickjjoubert) in [this great playlist](https://github.com/frederickjjoubert).

## Build & Run:

###  Install Rust

```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

### Build & Run

```
cd ./bevy-ball-game
cargo build
cargo run
```
